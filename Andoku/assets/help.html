Objective

The objective of the game is to fill the entire game grid so that the digits from 1 to 9 appear exactly once in each region. For a standard sudoku a region is either a row, a column or one of the nine square 3x3 blocks.

Game Variants

Andoku offers additional sudoku variants besides the classical one. In the New Game menu you can choose from two grid styles and four kinds of extra regions.

Grid Styles

The available grid styles are:

Standard
The grid is made up of square 3x3 blocks.
Squiggly (a.k.a. jigsaw)
Instead of 9 square blocks the game grid consists of 9 non-square regions. The regions are outlined with a thick line and the digits 1 to 9 have to appear exactly once in each region.

Extra Regions

Extra regions impose additional constraints on the puzzle. Aside from the classic version there are three kinds of extra regions:

None
You only have to deal with nine rows, columns and (square or squiggly) blocks.
X-Sudoku
The digits 1 to 9 also have to appear exactly once in the two diagonals of the puzzle. To indicate this the diagonals have a light blue background color.
Hyper-Sudoku
The sudoku has four additional regions drawn in a light blue background color. In addition to rows, columns and blocks the digits 1 to 9 have to appear exactly once in each of these extra regions.
Percent-Sudoku
The sudoku has three additional regions that resemble a percent sign.

Difficulty Levels

Andoku offers five difficulty levels per game type. The difficulties range from Easy to Fiendish. Squiggly sudokus or sudokus with extra regions are generally more difficult than standard sudokus.

The difficulty levels Hard and Fiendish should contain harder puzzles than what can be found in newspapers.

With eight different game types, five difficulty levels and 100 puzzles per type and difficulty Andoku offers a total of 4000 puzzles.

Input Methods

Andoku offers three different input methods for entering digits into the cells of a puzzle while playing. Go to Settings to choose between the available input methods.

Cell Then Values

With this input method you first select a cell by tapping on it with your finger and then you use the numeric pad at the bottom of the screen to enter one or more possible values for that cell. The currently selected cell will be highlighted with a background color.

Press the button C to clear the values of the currently selected cell.

The button Inv is used to reverse the values of the current cell. This makes it possible to easily specify that certain values are not possible at a certain cell. For example if you enter 3, 7, Inv the result will be 1, 2, 4, 5, 6, 8 and 9 (all digits except 3 and 7).

If the device has a track ball it can be used move the selection to one of the neighbor fields.

Values Then Cell

This input method works the opposite way around. First you select one or more values from the numeric pad at the bottom of the screen. Only then do you tap on a cell to assign these values to it.

This input method can save a lot of work because it is possible to assign multiple fields to the same value or values by tapping on the cells one after the other.

But what happens when you tap on a cell that is not empty? The result depends on whether the cell already contained one of the selected values or not. If the cell did not already contain one of the selected values the values are simply added (same behavior as if the cell would have been empty). If, on the other hand, the cell already contained one of the selected values then all of the selected values will be reset instead.

Example: cell values 3, numeric pad 5, 7 results in 3, 5 and 7.

Example: cell values 3, 5, 7, numeric pad 3, 5 results in 7.

Example: cell values 3, 5, numeric pad 5, 8 results in 3. I.e. 5 is cleared, but 8 is not set.

The button C is used to clear all values on the numeric pad. The button Inv is used to reverse the values on the numeric pad.

Automatic

Use this input method if you want to easily switch back and forth between Cell Then Values and Values Then Cell.

This means that initially both input methods are possible. If you tap on a cell first then Cell Then Values will be used automatically. If you tap on a digit of the numeric pad first then Values Then Cell will be used instead.

If you want to get back to the neutral state from Cell Then Values you have to tap on the selected cell a second time.

To get back from Values Then Cell you have to clear all values on the numeric pad. The simplest way to do this is by pressing the button C.
